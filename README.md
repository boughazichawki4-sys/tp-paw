# tp-paw

Projet de gestion des étudiants avec HTML, CSS, JavaScript et PHP.

## Description
Ce projet permet d'ajouter des étudiants dans un tableau et de gérer les présences et participations.

## Fonctionnalités
- Ajouter des étudiants
- Modifier les informations des étudiants
- Supprimer des étudiants
- Gérer les présences
- Consulter la liste des étudiants
